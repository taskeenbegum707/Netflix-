# 🎬 Netflix Clone (Frontend Project)

## 📌 Project Overview

This project is a **Netflix Landing Page Clone** built using **HTML, CSS, and JavaScript**.
It replicates the UI and basic interactions of the Netflix homepage, including trending sections, FAQs, and email validation.

---

## 🚀 Features

* 🎨 Responsive Netflix-style UI
* 📺 Trending Movies Section
* 📩 Email Validation with JavaScript
* ❓ FAQ Toggle (Open/Close functionality)
* 🔘 Smooth Scroll Effect
* 🔐 Sign-in Button Interaction (Demo alert)

---

## 🛠️ Technologies Used

* **HTML5** – Structure of the webpage
* **CSS3** – Styling and layout
* **JavaScript (Vanilla JS)** – Interactivity

---

## ⚙️ How to Run the Project

1. Download or clone this repository
2. Open the project folder
3. Double-click on `netflix.html`
4. Run in your browser 🚀

## 💡 Functionality Explained

### 📧 Email Validation

* Checks if the email field is empty or invalid
* Displays an alert message

### ❓ FAQ Section

* Click on a question to toggle the answer
* Uses JavaScript DOM manipulation

### 🔽 Smooth Scroll

* Clicking **Get Started** scrolls the page smoothly
